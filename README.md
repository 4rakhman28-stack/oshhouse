# OshHouse — Узбекская кухня на заказ

Современное веб-приложение для заказа узбекской еды с корзиной. Портфолио-проект.

**Живой сайт:** https://4rakhman28-stack.github.io/oshhouse/

## Как открыть локально

> Нельзя открывать `index.html` двойным кликом — страница будет пустой.

```bash
npm install
npm run dev
```

Открой в браузере: **http://localhost:5173/**

## Возможности

- Каталог из 14 блюд с фильтрацией по категориям
- Корзина с сохранением в localStorage
- Оформление заказа с формой доставки
- Адаптивный дизайн для мобильных устройств
- Продающий лендинг с hero-секцией и преимуществами

## Сборка

```bash
npm run build
npm run preview
```

## Стек

- React 19 + TypeScript
- Vite
- CSS (без фреймворков)
